# About Workspace Icons

These SVG icons are sourced from Iconify and stored locally so the app can embed
them through the GPUI asset source at build time.

| Local file | Iconify icon |
| --- | --- |
| `folder.svg` | [`material-symbols:folder`](https://icon-sets.iconify.design/material-symbols/folder/) |
| `markdown.svg` | [`mdi:language-markdown`](https://icon-sets.iconify.design/mdi/language-markdown/) |

The exported SVGs keep `fill="currentColor"` so the workspace tree can color the
icons using the active Velotype theme.
